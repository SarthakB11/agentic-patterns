# agentic-patterns

Twelve core agentic AI patterns, each implemented as a small, runnable, tested example. Every example runs offline against a deterministic mock provider, so you can clone the repo and run any pattern with no API key and no setup.

Status: under active construction. The full learning map lands here once all twelve patterns are in.

## Patterns

ReAct, Planning, Reflection, Tool use, Memory, RAG, Multi-agent orchestration, Evaluation loops, MCP, Guardrails, Human-in-the-loop, Routing.
